# Industriemontagen Leads App

Dies ist eine Streamlit-App zur Generierung von Leads für Industriemontagen-Dienstleister. Sie scannt Nachrichten auf Fabrikverlagerungen, Erweiterungen etc. in Europa und Asien (Deutsch/Englisch).

## Setup
1. Ersetze `API_KEY` in `app.py` mit deinem NewsAPI-Key (von newsapi.org).
2. Lokal testen: `streamlit run app.py`.
3. Für LinkedIn-Integration: Füge die LinkedIn API hinzu (siehe Kommentare im Code).



## Erweiterungen
- Integriere LinkedIn API für echte Kontakte.
- Automatisiere tägliche Scans mit einem Scheduler.
